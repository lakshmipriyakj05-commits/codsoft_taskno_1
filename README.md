# Personal Portfolio Website

## Overview

This project is a personal portfolio website built using HTML and CSS. It showcases personal information, skills, projects, resume, and contact details in a professional and responsive layout.

## Features

* Header section with name and introduction
* About Me section with profile image and bio
* Skills section
* Projects showcase section
* Resume download option
* Contact information section
* Footer with copyright details

## Technologies Used

* HTML5
* CSS3

## Project Structure

portfolio/
│
├── index.html

Note:
The project uses both HTML and CSS. The CSS code is embedded within the
`<style>` tag inside the `index.html` file instead of a separate `style.css` file.
Therefore, GitHub may display the repository language as 100% HTML even though
CSS is also used in the project

## How to Run

1. Download or clone the repository.
2. Open `index.html` in any web browser.

## Learning Outcomes

* Creating webpage layouts using HTML
* Styling websites with CSS
* Using images, links, and sections effectively
* Building a professional portfolio website
